#  Retail Sales Analysis (Power BI Project)

##  Project Overview

This project analyzes retail sales data using Power BI to uncover insights about sales performance, customer behavior, and product trends.

---

##  Objectives

* Analyze total sales and order trends
* Identify top-performing products and categories
* Understand customer segments and behavior
* Track sales performance by location and time

---

##  Dashboard Features

* KPI Cards: Total Sales, Orders, Customers, Avg Rating
* Sales Trend Over Time (Line Chart)
* Top Performing States (Bar Chart)
* Sales by Product Category (Pie Chart)
* Customer Segment Analysis

---
